miniAudicle
Windows
version 1.2.3 (Gidora)

Binary Release Installation Instructions
----------------------------------------
Simply drag the miniAudicle.exe application to an appropriate
directory, for example, C:\Program Files\miniAudicle\.  No
other support files are necessary.  

Bugs
----
Currently, it is not possible to stop and restart the virtual
machine.  Additionally, miniAudicle User Interface Elements
are not presently implemented.  

On Windows Vista, it is necessary to run miniAudicle in Windows
XP emulation mode.
