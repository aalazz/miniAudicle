The Qt build of miniAudicle is configured to link against the QScintilla library.

static qscintilla2_qt6 libraries; 
  qscintilla2_qt6.lib (release library)
  qscintilla2_qt6d.lib (debug library)
  qscintilla2_qt6d.pdb (debug symbols)

OR

dynamic qsctintilla2_qt6 libraries:
  qscintilla2_qt6.lib (release library)
  qscintilla2_qt6.dll (release DLL)
  qscintilla2_qt6d.lib (debug library)
  qscintilla2_qt6d.dll (debug DLL)
  qscintilla2_qt6d.pdb (debug symbols)

These library files can be either compiled from source or downloaded as pre-compiled
library files. The latest builds of the above can be found here:
  https://drive.google.com/drive/folders/1VC9vi5NaC9LvTt7GtacjiOpF34txjwoT
